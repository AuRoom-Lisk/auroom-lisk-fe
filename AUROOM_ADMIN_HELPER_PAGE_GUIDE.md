## 1. Overview

### 1.1 Purpose

Admin Helper Page adalah halaman utility untuk:
- **Testing**: Mint tokens, manage liquidity, test functions
- **Demo**: Simulasi yield, showcase features untuk hackathon
- **Debugging**: Check balances, allowances, contract states
- **Administration**: Manage KYC, inject yield, control access

### 1.2 Feature Summary

| Tab | Features | Access Level |
|-----|----------|--------------|
| **Faucet** | Quick faucet, individual mint, MNT link | Everyone |
| **Liquidity** | Pool overview, add/remove LP | Verified Users |
| **Vault** | Stats, yield injection, presets | Admin Only |
| **Identity** | Check, register, remove, batch | Admin Only |
| **Info** | Contract addresses, network info | Everyone |
| **Debug** | Balances, allowances, calculator | Everyone |

### 1.3 Total Helper Functions

| Category | Count |
|----------|-------|
| Token Operations | 5 |
| Liquidity Operations | 4 |
| Vault Operations | 4 |
| Identity Operations | 6 |
| Info/Debug Operations | 6 |
| **Total** | **25** |

type Role = 'guest' | 'connected' | 'verified' | 'admin' | 'owner';

const DEPLOYER = "0x742812a2Ff08b76f968dffA7ca6892A428cAeBb1";

function useUserRole(): Role {
  const { address, isConnected } = useAccount();
  const { isVerified } = useIsVerified(address);
  const { isAdmin } = useIsAdmin(address);
  
  if (!isConnected) return 'guest';
  if (address === DEPLOYER) return 'owner';
  if (isAdmin) return 'admin';
  if (isVerified) return 'verified';
  return 'connected';
}
